# Bundled Material assets

The runtime assets are fetched by `scripts/fetch_assets.ps1` and are kept as
ordinary files so applications can decide whether to install, embed, or replace
them.

- `Roboto-Light.ttf`, `Roboto-Regular.ttf`, `Roboto-Medium.ttf`: Roboto static
  fonts mirrored by the referenced `qt-material-widgets` project. Apache-2.0;
  see `fonts/LICENSE-Roboto.txt`.
- `MaterialIcons-Regular.ttf`: Google's Material Icons font. Apache-2.0; see
  `fonts/LICENSE-MaterialIcons.txt`.

Run `powershell -ExecutionPolicy Bypass -File scripts/fetch_assets.ps1 -Force`
to refresh the files from their upstream URLs.

The checked-in `src/embedded_fonts.cpp` contains the memory-optimized embedded
Regular/Icon pair generated by `scripts/generate_embedded_fonts.ps1`. Re-run
that script after refreshing the TTF files. Light/Medium are conditionally
compiled only when `IMGUI_MD2_EMBED_FULL_FONTS` is defined.
